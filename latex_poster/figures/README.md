# Figures Directory

Place your figure files here (PDF, PNG, JPG formats work best).

## Recommended formats:
- **PDF**: Best for vector graphics (logos, diagrams)
- **PNG**: Good for screenshots and raster images
- **JPG**: Good for photographs

## To include figures in the poster:

```latex
\includegraphics[width=0.8\textwidth]{figures/your_figure_name.pdf}
```

## Naming conventions:
- Use descriptive names: `funder_comparison.pdf`, `trend_analysis.png`
- Avoid spaces in filenames
- Keep extensions lowercase
