# Funder Analysis

Scripts for mapping and analyzing funding sources in biomedical research articles.

## Scripts

- `funder-mapping-parquet.py` - Maps articles to 31 major biomedical funders
- `funder_mapping.py` - Core funder mapping logic
- `funder-line-graph_v15.py` - Funder visualization tools

## Tracked Funders

10 major international funders + 22 NIH institutes:
- NIH, NSFC, European Commission, Wellcome Trust, MRC, NHMRC
- NSF, HHMI, DFG, Cancer Research UK
- Plus all 22 NIH institutes (NCI, NHLBI, NIMH, etc.)
