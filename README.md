# Open Science Metrics - Development Branch

> **Note:** This is the development branch containing work-in-progress code and internal documentation. It is not curated for external users.
>
> **For the public release, please see the [`main` branch](https://github.com/nimh-dsst/osm-2025-12-poster-incf/tree/main).**

---

## About This Branch

The `develop` branch contains:
- Development scripts and exploratory analysis code
- Internal documentation with environment-specific paths
- Intermediate results and experimental features
- HPC job scripts for NIH Biowulf cluster

This branch is intended for the NIMH Data Science and Sharing Team development workflow and may contain:
- Uncommitted experimental changes
- References to internal file systems
- Scripts that require specific HPC environments

## For External Users

Please use the **main branch** which contains:
- Curated poster materials for EBRAINS Summit 2025
- Clean analysis scripts with documented inputs/outputs
- Summary result CSVs ready for reproduction
- Public-facing documentation

**Main branch:** https://github.com/nimh-dsst/osm-2025-12-poster-incf/tree/main

## Branch Structure

| Branch | Purpose | Audience |
