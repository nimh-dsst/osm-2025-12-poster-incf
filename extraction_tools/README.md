# Extraction Tools

XML metadata extraction scripts for PubMed Central Open Access corpus.

## Scripts

- `extract_xml_metadata.py` - File-based extractor (229 files/sec)
- `extract_from_tarballs.py` - Streaming extractor (1,125 files/sec)
- `populate_metadata_iterative.py` - Metadata population from rtrans output
- `merge_parquet_*.py` - Parquet merging utilities
