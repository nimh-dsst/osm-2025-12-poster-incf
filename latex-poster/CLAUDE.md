# CLAUDE.md - LaTeX Poster Directory

This directory contains the LaTeX source files for poster #8 at [EBRAINS Summit 2025](https://summit2025.ebrains.eu/posters) on data sharing trends by funder.

The poster PDF was compiled using Overleaf.

## Directory Structure

```
latex-poster/
├── poster.tex              # Main poster document
├── preamble.tex            # LaTeX preamble with packages and styling
├── funder_table.tex        # Auto-generated funder data table
├── 2025-incf-brussels.bib  # BibTeX references
├── figures/                # PNG/PDF figures for the poster
│   ├── openss_funder_counts_by_year_v2.png
│   ├── openss_funder_percentages_by_year_v2.png
│   ├── EBrains.png
│   ├── NIH.png
│   └── DSST.png
└── OSM_INCF_Poster_Brussels_2025_12.pdf  # Compiled poster
```

## File Descriptions

### poster.tex
Main document containing:
- Title, authors, affiliations
- Introduction, Methods, Discussion, Takeaways sections
- Results section with figures and funder table
- Uses TikZ-based custom poster layout

### preamble.tex
Contains:
- Package imports (tikz, xcolor, siunitx, booktabs, biblatex)
- Color definitions (COL1-COL7, COLtitle)
- Custom `\posterbox` command for poster sections
- Custom `poster` environment with header/footer layout
- siunitx configuration for number formatting

### funder_table.tex
Auto-generated table with:
- 47 funders sorted by data sharing percentage
- Conditional formatting (blue-white-red color scale)
- siunitx S columns for numeric alignment
- Generated by `analysis/funder_table_latex.py`

### 2025-incf-brussels.bib
BibTeX references including:
- Serghiou2021-pl (rtransparent)
- Riedel2025-qq (OddPub)
- Hamilton2023-yz (data sharing meta-analysis)
- Iarkaeva2024-wo (DAS sensitivity)
- ThomasUnknown-lc (this repository)

## Regenerating the Funder Table

```bash
cd ..  # Back to osm-2025-12-poster-incf
python analysis/funder_table_latex.py \
    --input results/funder_data_sharing_summary_v3_all.csv \
    --aliases funder_analysis/funder_aliases_v3.csv \
    --sort-by data_sharing_pct \
    --color-pubs \
    --color-pct \
    --output latex-poster/funder_table.tex
```

## Key Poster Content

- **Corpus**: 6.7M publications from PMC Open Access (June 2025)
- **Method**: rtransparent + OddPub v7.2.3 for open data detection
- **Finding**: ~6.8% of research articles have open data statements
- **Funders**: 57 canonical funders identified via NER
- **Year range**: 2010-2024
